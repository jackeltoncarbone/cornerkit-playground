# CornerKit — Playground

iOS-grade superellipse corners as an SVG `clip-path`, with Pill, Circle, Scoop and Notch modes. Every tile stacks `backdrop-filter` glass + a cast shadow + an SVG rim on a squircle clip.

**Live:** https://jackeltoncarbone.github.io/cornerkit-playground/

Single self-contained `index.html` — no build, no deps.
